# SpaceX Tailwind Clone

A responsive SpaceX-inspired landing page clone built with **HTML** and **Tailwind CSS** as a learning project. This project recreates key design elements of the SpaceX website, including full-screen video backgrounds, responsive navigation menus, dropdowns, and mobile-friendly layouts.

## Live Demo

https://madhushreerasal.github.io/Spacex-Tailwind-Clone/

## 📸 Features

- Responsive design for desktop and mobile devices
- Full-screen video hero sections
- Mobile navigation menu built without JavaScript
- Hover-based dropdown menus
- Background images and video sections
- Tailwind CSS utility-first styling
- SpaceX-inspired layout and content

## Built With

- HTML5
- Tailwind CSS
- Remix Icon

## What I Learned

This was my first project using Tailwind CSS. While building it, I practiced:

- Tailwind utility classes
- Responsive design using breakpoints
- Flexbox layouts
- Positioning elements with Tailwind
- Creating dropdown menus with `group` and `peer`
- Working with video backgrounds
- Mobile-first development concepts
